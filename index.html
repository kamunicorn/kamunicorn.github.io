Ирвас окна

Сайт «Ирвас окна» - мой дипломный проект на курсе «Javascript» Академии верстки. Мне был дан полностью сверстанный сайт, сделанный адаптивно. Моей задачей было добавить интерактив сайту в соответствии с требованиями технического задания. Было реализовано:

  • - появление модальных окон с формой обратной связи;
  • - отправка данных пользователя через несколько форм обратной связи;
  • - два слайдера с информацией;
  • - таймер обратного отсчета;
  • - калькулятор расчета стоимости с отправкой данных для расчета.

Процесс работы над сайтом велся в сервисе Trello.

Информационная система предприятия

Доработка информационной системы предприятия с разработкой новых модулей на основе языков javascript и php:

  • - авторизация пользователя (2 уровня доступа);
  • - редактирование таблицы базы данных из браузера с использованием плагина на jQuery;
  • - SQL-запрос формируется динамически на основе данных от пользователя;
  • - получение данных от сервера приходят в формате json;
  • - рисование таблицы на основе полученных данных с учетом содержания этих данных (выделение цветом некоторых строк и т.д.)

За более подробной информацией и скриншотами просьба обращаться в личном сообщении.
