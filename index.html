Ирвас окна

Сайт «Ирвас окна» - мой дипломный проект на курсе «Javascript» Академии верстки. Мне был дан полностью сверстанный сайт, сделанный адаптивно. Моей задачей было добавить интерактив сайту в соответствии с требованиями технического задания. Было реализовано:

  • появление модальных окон с формой обратной связи;
  • отправка данных пользователя через несколько форм обратной связи;
  • два слайдера с информацией;
  • таймер обратного отсчета;
  • калькулятор расчета стоимости с отправкой данных для расчета.

Итоговый вариант построен на модульной структуре (Webpack), подключена сборка проекта. Процесс работы над сайтом велся в сервисе Trello.

Информационная система предприятия

Доработка информационной системы предприятия с разработкой новых модулей на основе языков javascript и php:

  • авторизация пользователя (2 уровня доступа);
  • редактирование таблицы базы данных из браузера с использованием плагина на jQuery;
  • SQL-запрос формируется динамически на основе данных от пользователя;
  • получение данных от сервера приходят в формате json;
  • рисование таблицы на основе полученных данных с учетом содержания этих данных (выделение цветом некоторых строк и т.д.)

Балашиха хлеб>

Верстка по дизайн-макету в формате psd (макет можно посмотреть здесь). Был реализован следующий функционал:

  • появление модальных окон с дополнительной информацией;
  • отправка данных пользователя через три формы обратной связи;
  • слайдер с фотографиями;
  • подключена Яндекс.Карта с оптимизацией загрузки (увеличивает скорость загрузки сайта);
  • отправка данных пользователя (введенных в форме обратной связи) на почту с помощью языка php.

Сайт был сверстан адаптивно и хорошо выглядит как на больших мониторах, так и на смартфонах. Также сайт одинаково хорошо смотрится и работает во всех современных браузерах, в том числе Internet Explorer 11, Edge и Safari (Mac OS). Была проведена оптимизация скорости загрузки (проверка с помощью Google Page Speed Insights) с повышением показателей загрузки.

Итоговый вариант построен на модульной структуре (Webpack), подключена сборка проекта.
