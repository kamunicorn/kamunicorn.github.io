    Применяется в всплывающем окне для памяток. Для нагдядности обернуты в контейнер с шириной, высотой и классом .relative

  • ul.ui-dotslist > li > button.ui-dot

    Используется на главной в слайдере

    Цвет активного дотса задается атрибутом [bg-active="secondary"], который активируется, когда у дотса присутствует класс .active

  • ul.ui-steplist[role="tablist"]
    > li > button.ui-step[role="tab"]

    Используется в панели настроек для слабовидящих (ограничивающий контейнер для примера)

    Не пройденный шаг

    button.ui-step[role="tab"]

    Пройденный шаг

    button.ui-step.active[role="tab"]

    Текущий шаг

    button.ui-step.active[aria-current="true"][role="tab"] 
  • ul.ui-tablist[role="tablist"] > 
    li > button.ui-tab[role="tab"] > .ui-tab-triangle

    Используется на многих страницах.

    Клавиши влево и вправо для выбора вкладки, клавиша вниз для перехода к содержимому вкладки

    Цвет активной вкладки задается атрибутом [bg-active="secondary"], который активируется, когда у вкладки присутствует класс .active

  • a.ui-icon-link

    Используется в таблицах на страницах Результаты / Получить результаты и ГИА-9(ГИА-11) / Даты объявления результатов и подачи апелляций

    Пример со словами (иконки имеют внешний отступ margin-left):
    Слово
    19.05.2019
  • a.ui-grade-note

    Используются в таблице на с. "ГИА / Информация / Минимальные баллы", имеют отступ margin-left

    С помощью aria-labelledby="id" должны быть связаны с примечанием (оно будет озвучено при фокусе на ссылку)

    ! примечание 1 
  • .ui-accordion-open__wrapper > 
    button.ui-accordion-open

    Пока применяется только в style guide, ширина 100%.

    В элементе .ui-accordion-open может быть что угодно. Для наглядности компонент вложен в обертку. Также, для удобства блоку-обертке .accordion__block.js-accordion-block устанавливается класс .is-open

    Описание взаимодействия

    При открытии кнопке ставится атрибут [aria-expanded="true"], при закрытии и первой инициализации false.
